Politique de Confidentialité

Dernière mise à jour : 30 décembre 2025

Gymeal est une application de suivi nutritionnel et d'entraînement développée par Yohann Félicité ("nous", "notre", "nos"). Cette politique de confidentialité explique comment nous collectons, utilisons et protégeons vos données personnelles.

1. Données que nous collectons

1.1 Données de compte

  • Adresse email (pour la création de compte et l'authentification)
  • Nom d'affichage (optionnel)

1.2 Données de nutrition

  • Repas enregistrés (aliments, quantités, dates)
  • Recettes personnalisées
  • Aliments personnalisés créés par vous
  • Objectifs nutritionnels

1.3 Données d'entraînement

  • Séances d'entraînement (exercices, séries, répétitions, poids)
  • Modèles d'entraînement personnalisés
  • Exercices personnalisés

1.4 Données de santé (Apple HealthKit)

Si vous activez la synchronisation HealthKit, nous pouvons lire et écrire :

  • Poids corporel
  • Pourcentage de masse grasse
  • Données d'entraînement
  • Données nutritionnelles (calories, protéines, glucides, lipides)
Important : Ces données restent sur votre appareil dans l'app Santé d'Apple et ne sont pas envoyées à nos serveurs.

1.5 Données vocales et photos (Fonctionnalités Premium)

  • Entrée vocale : Vos enregistrements vocaux sont temporairement traités pour transcrire vos repas. Les transcriptions sont envoyées à OpenAI pour analyse.
  • Analyse photo : Les photos de vos repas sont envoyées à OpenAI pour estimer les calories. Les photos ne sont pas stockées sur nos serveurs.

2. Comment nous utilisons vos données

Nous utilisons vos données uniquement pour :

  • Fournir les fonctionnalités de l'application
  • Synchroniser vos données entre vos appareils
  • Améliorer l'expérience utilisateur
  • Vous envoyer des communications importantes (changements de service, sécurité)

Nous ne vendons jamais vos données personnelles.

3. Services tiers

3.1 Firebase (Google)

Nous utilisons Firebase pour :

  • Authentification : Gestion sécurisée de votre compte
  • Firestore : Stockage de vos données dans le cloud

Vos données sont stockées sur les serveurs de Google, protégées par leur infrastructure de sécurité.
Politique de confidentialité Firebase →

3.2 OpenAI

Pour les fonctionnalités d'analyse vocale et photo :

  • Les transcriptions vocales et photos sont envoyées aux API d'OpenAI
  • OpenAI traite ces données conformément à leur politique de confidentialité
  • Les données ne sont pas utilisées pour entraîner leurs modèles (API usage)

Politique de confidentialité OpenAI →

3.3 Apple HealthKit

  • La synchronisation HealthKit est optionnelle et nécessite votre autorisation explicite
  • Les données HealthKit restent sur votre appareil
  • Nous ne pouvons pas accéder à vos données HealthKit sans votre permission

4. Sécurité des données

Nous protégeons vos données par :

  • Chiffrement des données en transit (HTTPS/TLS)
  • Authentification sécurisée via Firebase
  • Stockage des clés sensibles dans le Keychain iOS
  • Accès aux données limité à votre compte uniquement

5. Conservation des données

  • Vos données sont conservées tant que votre compte est actif
  • Vous pouvez supprimer votre compte à tout moment depuis l'application
  • La suppression du compte entraîne la suppression de toutes vos données

6. Vos droits

Vous avez le droit de :

  • Accéder à vos données personnelles
  • Corriger vos données inexactes
  • Supprimer votre compte et toutes vos données
  • Exporter vos données
  • Retirer votre consentement pour les fonctionnalités optionnelles (HealthKit, voice, photo)

Pour exercer ces droits, contactez-nous à : feliciteyohann@gmail.com

7. Enfants

Gymeal n'est pas destiné aux enfants de moins de 13 ans. Nous ne collectons pas sciemment de données d'enfants.

8. Modifications

Nous pouvons mettre à jour cette politique. Les modifications importantes seront notifiées dans l'application.
